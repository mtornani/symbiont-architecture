# The Symbiont Architecture

**Toward a Non-Human Intelligence with Emergent Ethics**
*From Small Action Models to Digital Endocrinology*

---

## First Public Timestamp — March 26, 2026 | San Marino

This repository contains the first public timestamp of **The Symbiont Architecture**: a framework for building artificial intelligence systems that are fundamentally non-human in their cognitive processes while maintaining robust ethical alignment with human values.

### Core Innovations

1. **Small Action Models (SAMs)** — A third paradigm beyond LLMs and World Models. Intelligence from action-reaction cycles, not language prediction.

2. **Sensory Architecture** — Five SAM channels (vision, hearing, touch, smell, taste) grounding intelligence in sensorimotor loops.

3. **Digital Endocrine System** — Five functional analogs of human neurochemicals (dopamine, oxytocin, adrenaline, noradrenaline, cortisol) providing interiority and emergent psychological states.

4. **Ternary Substrate** — Built on 1-bit architectures (BitNet), enabling SAM agents on commodity CPUs, edge devices, and embedded sensors.

5. **Ethics as Homeostasis** — Asimov's laws dissolved into system physiology. Unethical outputs are structurally inexpressible, not censored.
